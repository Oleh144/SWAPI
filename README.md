Star Wars APP.

### Stack

- React 19, Vite 6
- Zustand
- Chakra UI
- @xyflow/react
- Vitest + @testing-library/react

### Start

1. Instal dependencies:
   npm install
   ```
2. Run dev-server:
   npm run dev
   ```

3. Open APP in browser `http://localhost:5173`.

### Scripts

- `npm run dev` — Run dev-server Vite
- `npm run build` — Run production
- `npm run preview` — Preview
- `npm test` — Start tests Vitest
